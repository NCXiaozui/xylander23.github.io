处理数据.md

发布于 2019-12-19 CS  C++ 

[toc]
C++数据类型可以分为两种，一种是基本类型，一种是复合类型，基本类型主要又分为整数和浮点数两种。本篇主要介绍是基本型。

变量名

C++有自己的变量名规范：

  1. 在名称中只能出现字符、数字和下划线
  2. 名称的第一个字符不能是数字
  3. 区分大写字符与小写字符
  4. 不能将C++关键字作为变量名
  5. 以两个下划线或者下划线和大写字母打头的名称被保留给编译器使用，一个下划线开头的作为全局标识符。

变量初始化

可以通过等号的方式： type v_name = value;或者 type v_name{}

整数型

short\int\long\long long

整数型主要分为short, int, long,以及新加的long long，这个顺序也是按从小到大的顺序排:short(16bit) <= int <= long(32bit) <= long long(64bit)
但是要注意的是，不同的操作系统，这些变量占的内存大小不是一定的，但是是至少要满足上面关系，同时可以通过climits来查看。
（1字节byte = 8 位bit）
sizeof()返回类型或变量的长度，单位是字节。
以上是表示有符号的，如果是无符号的，那么就可以带unsigned方式。
一般整数都是十进制的，如果是前面带了0第二位又是1~7就表示是八进制，如果是0x就是16进制。
输出8进制或者是16进制要先改变一下输出流，在cout之前，先输出 hex或者oct。

常量类型

通常C++会把整型常量存储为int，然后会看后缀，如果后缀带l或L，就是long常量，如果u或者U就是unsigned int, ul 表示unsigned long常量，以此类推。
十六制通常用unsigned int来表示。

char

char是字符8位字节，但是它的位数比short还少，而且实质也是一个数字，是可以直接进行加减这些操作的。（可以乘除吗？如果超过了范围输出什么？）
如果输出的超过了，可以用wcha_t,相应的输入输出要用wcin和wcout

bool

只有两个值，1或0分别表示真或者假

const

常量限定符，用了const以后就不能再修改了。
方式：const type name = value;
特别说明一下，定义常量的时候一定要初始化，不能你就不能再修改赋值了。

浮点型

浮点型是因为有两个部分组成，一个是值，一个是表示对值放大还是缩小多少。因为是靠移动小数点的，所以术语叫浮点，也因此得名。

浮点的写法

浮点写法有两种：

  1. 使用标准的小数点表示：
    8.23
    2.用科学记数法：
    2.52E+(-)8

float/double/long double

C++有三种浮点类型，从小到大：float(32)<\double(64)<\long double(80,96)

浮点常量：

可以在数字后面带上f/F(float),l/L(long double)，不带就是double
比如1.23f

C++算术运算符：

C++的优先级和数学上面是保持一致的。

除法分支

除法要注意，如果两个数都是整数，那么结果也是整数，而且不是四舍五入，如果有一个是浮点，那么就是浮点型，而且是向最高的转变。

类型转换

1.初始化和赋值进行的转换：
C++允许将一种类型的值赋给另一种类型变量，值将会被转化为的类型，如果值大于被转化的类型，将可能失真。
2.表达式转换
整型提升
在计算表达式中，C++将bool,char,unsigned char, singed char和short值转换为int。

在最后一行，ducks和chickens会先转换成int，然后把结果转成short,因为fowl是short。
当运算涉及两种类型，较小的类型会被转换为较大的类型。
特别说下遇到有符号和无符号的情况：
如果一个操作数有符号，一个无符号，且无符号操作数的级别比有符号操作数级别高，那么有符号转成无符号的类型，
否则，有符号类型可表示无符号类型的所有值，则无符号操作数转换为有符号操作类型，
否则，将两个操作数都转换为有符号类型的无符号版本。
整型级别：long long > long > int > short > signed char (signed char\ unsigned char)> bool

  1. 传递参数时的转换
    传递参数的类型转换通常由C++函数原型控制。C++将对char和short类型(signed和unsigned)应用整型提升，另外为保持与传统C语言大量代码兼容性，在将参数传递给取消原型对参数传递控制的函数时，C++会对float提升到double.
